Cadence is **emails per month** for one audience group. The planner converts it into days between emails, so fractions work exactly as you expect.

## What the numbers mean
| Cadence / month | Days between emails | In practice |
|---|---|---|
| 4 | ~8 days | weekly-ish |
| 2 | ~15 days | every two weeks |
| 1.5 | ~20 days | every three weeks |
| 1 | ~30 days | monthly |

Type `1.5` or `1,5` — both work; the comma is converted automatically.

## Where to set it
- **For one plan run:** the "Cadence / month" box on the [Director page](/university/newsletter/newsletter-plan-a-month), next to the date range. This value is used for the plan you generate right now.
- **Per-group default:** Settings → Property → Audience → [Group cadence panel](/university/newsletter/newsletter-audience-settings). This is the group's standing rhythm.

:::tip Why cadence matters
Too many emails and guests unsubscribe; too few and the hotel is forgotten between trips. B2B groups (DMCs, retreat hosts) usually run at 1–1.5 per month; guest groups can run higher in high season.
:::

Expected outcome: after generating a plan, the gaps between the group's slots on the calendar match the table above (slots always land on Tuesdays or Thursdays, so the exact gap can shift by a day or two).
