# Brand: Donna Portals

Hospitality portal / booking platform with beach club concept. Premium positioning.

## Positioning

- **Premium beach club**, not budget
- **High-end repositioning** — never compete on price
- Focus on experience, ambiance, exclusivity

## Visual

- **Color scheme**: black + dark green (especially for presentations)
- **Bali beds** prominent in package visuals — they're the signature element
- **Photography**: sunset beach scenes, Bali bed setups, premium F&B, lifestyle
- **Format**: landscape preferred for presentations

## Voice

- Confident, not boastful
- Specific over generic ("private cabana with sunset view" not "luxury setup")
- Never apologetic, never discount-led
- Avoid "exclusive" overuse — show, don't tell

## Partnership structure

- 50/50 profit split (after cost recovery)
- Equity option after 12 months
- Documented in revenue framework master doc
- Never modify partnership terms in code/comms — PBS handles

## Marketing rules

- No flash sales, no "today only" pressure tactics
- Photos must show Bali beds in package promos
- Never quote competitors directly
- Press / influencer requests reviewed by PBS before response

## Booking experience

- Premium handoff feel — no generic checkout
- Confirmation includes personal touch (host name, what to expect)
- Pre-arrival: arrival logistics + curated suggestions
- On-site: Bali bed reserved with name card

## What never happens

- Discount language in main copy
- Generic "beach vacation" framing — must be specific to Donna
- Comparison with budget OTAs
- Auto-generated packages without curatorial review

## Action items for agents

When working on Donna Portals:
- Check this file before any copy, package, or visual change
- Black + dark green for presentations (no other accent colors without ADR)
- Bali beds always present in package imagery
- Never suggest a price-led promo — propose value-add instead (extra night, F&B credit, transfer)
