# 07 — SOP Library

> Index of all staff-facing SOPs. Format rules + status per SOP.

## Format rules (all SOPs)
- Visual-first: ≥70% imagery, ≤30% text.
- Dual language: Lao primary, English secondary.
- Max 1 action per page/slide.
- Color-coded by department (see `11_BRAND_AND_UI_STANDARDS.md`).
- Numbered steps, large icons, real Cloudbeds screenshots.
- File naming: `SOP-[DEPT]-[NUM]-[short-name]-v[X].pdf`

## Production workflow
1. Capture screenshots from live Cloudbeds (sanitized).
2. Draft in Figma → review with department head.
3. Translate to Lao (native reviewer required).
4. Print laminated quick cards for stations.
5. Embed in Cloudbeds onboarding for new hires.
6. Re-review every 6 months or on Cloudbeds UI change.
